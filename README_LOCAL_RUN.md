# Run locally on macOS

## Prereqs
- Install Node.js LTS

## Start
```bash
cd Nexus-Enterprise-Suite-ready
npm install
npm run dev
```

Then open the localhost URL shown by Vite.

## Demo login
- Email: `admin@erp.com`
- Password: `admin123`

## Notes
- This app uses browser `localStorage` as its data store.
- To reset demo data, clear site data/local storage for the local dev URL.
